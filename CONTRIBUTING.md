# Contributing

## Suggest New Tool Ideas

Open a tool idea issue with:

- the workflow stage it protects
- the input it reads
- the output it produces
- the safety risks it avoids
- whether it needs network access or credentials

## Contribute Docs

Docs contributions are welcome when they clarify safe AI coding agent workflows, realistic limitations, and human review boundaries.

## Propose Shared Safety Rules

Shared safety rules should be conservative, testable, and explainable. They should not depend on private account data or hidden services.

## Do Not Add

- telemetry
- cookie, token, password manager, or keychain access
- auto-claim logic
- payment, payout, withdrawal, tax, or KYC logic
- spam PR automation
- exploit instructions
