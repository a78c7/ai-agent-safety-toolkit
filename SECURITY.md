# Security

This toolkit avoids credential access by design.

## Boundaries

- no cookie, keychain, token, or password manager access
- no secret upload
- no automatic bounty claim
- no KYC, payment, tax, or withdrawal handling
- no GitHub Sponsors setup
- human review required

## Reporting Safety Concerns

Report safety concerns via GitHub issues or GitHub security advisories.

Do not report exploit instructions, credential material, cookies, tokens, private keys, or private account data.

This project does not handle bounty payment, platform payout, KYC, tax, or withdrawal workflows.
