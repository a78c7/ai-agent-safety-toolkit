# Long Announcement

AI Agent Safety Toolkit is a small umbrella project for three safety-first tools around AI coding agent workflows.

The workflow is:

```text
GitHub issue
-> BountyLens
-> Testability Doctor
-> Codex / Claude Code / Cursor
-> Tests
-> AgentGate
-> Draft PR
-> Human review
```

The tools answer three practical questions:

- BountyLens: does this bounty issue look worth attempting?
- Testability Doctor: does this repo look practical for an AI agent to modify and test?
- AgentGate: does this generated diff look safe enough to move toward a PR?

This is not an automatic earning system, not a PR spam tool, and not a replacement for maintainers.

It does not read cookies, tokens, keychains, or password managers. It does not handle payment, KYC, payout, or withdrawal flows. It does not claim bounties.

The goal is to make AI-assisted code work more explicit before humans have to review it.
