# Comparison Table

| Question | Tool |
|---|---|
| Is this bounty issue real? | BountyLens |
| Is this repo testable by an AI agent? | Testability Doctor |
| Is this AI-generated diff safe for PR? | AgentGate |
| Should I claim a bounty? | Human only |
| Should I handle KYC/payment? | Human only |
| Should this PR be merged? | Maintainer |

## Notes

The tools provide signals and guardrails. They do not decide reward claims, account steps, payouts, or merge decisions.
