# Maintainer Pitch

AI Agent Safety Toolkit is for maintainers who want clearer boundaries around AI-assisted contributions.

## How to Use It

Use AgentGate to add a safety check for AI-generated PRs.

Use Testability Doctor to document which kinds of contributions are practical to test in your repository.

Use BountyLens only if your project participates in bounty platforms or receives bounty-driven contributors.

## Why It Helps

The toolkit encourages contributors to:

- understand issue risk before starting
- check whether the repo has a practical test path
- keep AI-generated diffs narrow
- open draft PRs
- include test and risk notes

## What It Does Not Do

It does not replace maintainer review, merge judgment, project policy, or security review.

It does not handle KYC, payment, payout, withdrawal, or bounty claims.
