# Short Announcement

AI Agent Safety Toolkit is a safety-first workflow for AI coding agents:

```text
BountyLens -> Testability Doctor -> AgentGate
```

It helps check bounty issue risk, repository testability, and AI-generated diffs before draft PR and human review.

It does not claim bounties, handle payment/KYC, read cookies/tokens/keychains, or replace review.
