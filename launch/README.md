# Launch Story Pack

This folder contains launch drafts for AI Agent Safety Toolkit.

Included:

- Blog post
- Social posts
- Case study
- Maintainer pitch
- Developer pitch
- Comparison table
- FAQ
- Checklist
- Launch workflow diagram

These are publishing drafts. They are not posted automatically.

## Files

- `launch-blog-post.md`: long-form launch article
- `short-announcement.md`: compact announcement copy
- `long-announcement.md`: longer project announcement
- `social-posts.md`: X/Twitter, LinkedIn, Reddit, and Hacker News drafts
- `case-study-ai-agent-pr.md`: realistic workflow case study
- `comparison-table.md`: quick question-to-tool table
- `maintainer-pitch.md`: message for open-source maintainers
- `developer-pitch.md`: message for developers using AI coding agents
- `faq-for-launch.md`: launch-specific FAQ
- `launch-checklist.md`: manual launch checklist
- `launch-workflow.svg`: visual launch workflow diagram
