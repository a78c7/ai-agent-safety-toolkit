# Quickstart

## Bounty Workflow

```text
GitHub bounty issue
-> BountyLens
-> Testability Doctor
-> Codex / Claude Code / Cursor
-> Project tests
-> AgentGate
-> Draft PR
-> Human review
```

## Non-Bounty Workflow

```text
Repository issue
-> Testability Doctor
-> AI coding agent
-> Project tests
-> AgentGate
-> Draft PR
-> Human review
```
