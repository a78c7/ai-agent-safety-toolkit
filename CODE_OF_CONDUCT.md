# Code of Conduct

## Expected Behavior

- Be respectful and practical.
- Keep discussion focused on safety-first AI coding workflows.
- Avoid sharing private credentials or account data.
- Use evidence when proposing detector or workflow changes.

## Unacceptable Behavior

- Harassment or personal attacks.
- Posting secrets, tokens, cookies, or private keys.
- Encouraging unsafe automation around payments, KYC, or account access.
- Encouraging spam pull requests.

## Enforcement

Maintainers may close issues, reject changes, or remove content that violates this code of conduct.
